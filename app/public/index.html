Web Font Optimization for Mobile Internet Users

A performance study of resource prioritization approaches for optimizing custom fonts on the web.

Compare them by testing them locally, with the network throttling profile set to 4G. *Details about that here* and reveal the difference in loading speed.

The typeface that was used is called Source Sans Pro.

You need a SSL certificat for localhost to be able to test the strategies that are using HTTP/2 Server Push. Certificats are included, but if they have expired please follow this guide.
